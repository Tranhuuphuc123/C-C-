>>>> KIẾN THỨC TỔNG QUÁT:
 - Chương trình c++ đầu tiên:
        #include<iostream>
        using namespace std;
        int main(){
            // khối lệnh thực thi
            return 0;
        }
    
   + trong đó: 
      ++ thư viện c++: iostream.
      ++ using namespace std; là cách khai báo toán tử phạm vi (std) trong c++. Vì nếu không khai báo std
      (toán tử phạm vi trong c++) thì khi sử dụng các hàm khối lệnh trong c++ ta bắt buộc phải thêm std:: vào
      câu lệnh: vd( std::cout<<, std::cin>>....) thay vì chỉ cần viết cout<<, cin>> một cách đơn giản hơn nhiều.
